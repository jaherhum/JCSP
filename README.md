# JCSP
### Jaherhum's Clean Switch Pack

Welcome to the official JCSP distribution repository — a lightweight, curated pack built for a clean and reliable homebrew experience on modded Nintendo Switch consoles.

---

## Overview

JCSP (Jaherhum's Clean Switch Pack) is a carefully selected collection of essential homebrew tools, apps, and configurations, focused on stability and simplicity over bloat. This repository is the primary distribution channel for stable, tested releases.

## Features

- Minimal, clean setup with no unnecessary clutter
- Curated selection of essential homebrew apps
- Easy install and overwrite on existing setups

## Getting Started

1. Download the latest release from the [Releases](https://github.com/jaherhum/JCSP/releases) section.
2. (Optional) Back up your `/atmosphere` and `/bootloader` folders before installing.
3. Extract the contents directly to the root of your SD card.

> [!IMPORTANT]
> If you are updating Atmosphère or your system firmware, delete the existing `atmosphere` folder before copying the new files to avoid conflicts.

## Requirements

- Nintendo Switch with Atmosphère installed
- SD card with sufficient free space
- Any additional firmware or CFW requirements, if applicable
